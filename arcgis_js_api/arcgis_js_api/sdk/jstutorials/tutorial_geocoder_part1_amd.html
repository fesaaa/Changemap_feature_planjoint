Geocoder Widget Tutorial

Using the Geocoder Widget

Panning and zooming a map to get to the place you’d like to go can be time consuming and a pain in your wrist. One way to provide easier map navigation is by adding a search box. A search box lets users to jump to a location quickly and easily. Now it's easier to add a search box that does just that by using the Geocoder widget with the ArcGIS Global Geocoding service.

In this tutorial, you will add the Geocoder widget to a basic web map to allow your users to easily search for locations. We’ll go through a few customizations that can be applied to the Geocoder widget in the next tutorial.

View a live version of this sample in the ArcGIS API for JavaScript Sandbox.The sandbox is a live code editor that allows you to modify the sample's source and view the changes live.

Note: This tutorial shows how to use modules from the ArcGIS API for JavaScript using Asynchronous Module Definition(AMD).


Prerequisites

  • JavaScript experience
  • An IDE or code text editor
  • Some familiarity with Dojo (1.7 or later), the ArcGIS JavaScript API and ArcGIS Online.

Before Getting Started

  • Start with a simple web application with your a map defined.
  • Make sure you're using the latest version of the ArcGIS JavaScript API (3.5 or later)

Setting Up the Geocoder Widget

  1. Require the following widgets. 
       require(["esri/map", "dojo/dom", "esri/dijit/Geocoder", "dojo/domReady!"], function(Map, dom, Geocoder) {
     //code goes here

   });
  2. Create an element in your HTML document to house the widget. 
          <body class="soria">
        <div id="search"></div>
        <div id="map"></div>
      </body>
  3. Specify a style to position the widget over the map. 
       html, body, #map {
     padding: 0;
     margin: 0;
     height: 100%;
   }
   #search {
     display:block;
     position:absolute;
     z-index:3;
     top:20px;
     left:75px;
   }
  4. Define the Geocoder widget. As the second parameter, define the node where you want to attach the widget. We created this node already in our HTML document and named it search. This sample specifies the minimum required Geocoder parameters. 
      var geocoder = new Geocoder({
     map: map,
  },dom.byId("search"));
  5. Startup the widget. 
       geocoder.startup();

The Geocoder widget should now be fully functional.

The next tutorial, Geocoder Widget Tutorial - Part 2, covers customization of the Geocoder widget.
